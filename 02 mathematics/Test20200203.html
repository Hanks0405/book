4.2 심파이를 사용한 함수 미분¶

import sympy

# Juypter 노트북에서 수학식의 LaTeX 표현을 위해 필요함
sympy.init_printing(use_latex='mathjax')

3 dimensional Rosenbrock function

\[ f(x) = (1 - x_1)^2 + (1 - x_2)^2 + 100\left( (x_2-x_1^2)^2 + (x_3 - x_2^2)^2 \right) \]
x1, x2, x3 = sympy.symbols("x1 x2 x3")

f = (1 - x1)**2 + (1 - x2)**2 + 100*((x2-x1**2)**2 + (x3-x2**2)**2)

\(\dfrac{\partial f}{\partial x_1}\)

sympy.diff(f, x1)
\[\displaystyle - 400 x_{1} \left(- x_{1}^{2} + x_{2}\right) + 2 x_{1} - 2\]

\(\dfrac{\partial f}{\partial x_2}\)

sympy.diff(f, x2)
\[\displaystyle - 200 x_{1}^{2} - 400 x_{2} \left(- x_{2}^{2} + x_{3}\right) + 202 x_{2} - 2\]

\(\dfrac{\partial f}{\partial x_3}\)

sympy.diff(f, x3)
\[\displaystyle - 200 x_{2}^{2} + 200 x_{3}\]

\(\dfrac{\partial^2 f}{\partial x_1^2}\)

sympy.diff(f, x1, x1)
\[\displaystyle 2 \left(600 x_{1}^{2} - 200 x_{2} + 1\right)\]

\(\dfrac{\partial^2 f}{\partial x_2^2}\)

sympy.diff(f, x2, x2)
\[\displaystyle 2 \left(600 x_{2}^{2} - 200 x_{3} + 101\right)\]

\(\dfrac{\partial^2 f}{\partial x_3^2}\)

sympy.diff(f, x3, x3)
\[\displaystyle 200\]

\(\dfrac{\partial^2 f}{\partial x_1x_2}\)

sympy.diff(f, x1, x2)
\[\displaystyle - 400 x_{1}\]

\(\dfrac{\partial^2 f}{\partial x_2x_3}\)

sympy.diff(f, x2, x3)
\[\displaystyle - 400 x_{2}\]

\(\dfrac{\partial^2 f}{\partial x_3x_1}\)

sympy.diff(f, x3, x1)
\[\displaystyle 0\]

5.2 제한조건이 있는 최적화 문제¶

\[ \arg \max xy \]
\[ x^2 + y^2 \leq 1 \]
\[ y \geq 0 \]
def f(x):
    return -x[0] * x[1]

def c1(x):
    return 1 - (x[0] ** 2 + x[1] ** 2)

def c2(x):
    return x[1]


sp.optimize.fmin_slsqp(f, np.array([1, 1]), ieqcons=[c1, c2])
