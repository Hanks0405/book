def heaviside_step(x):
    if isinstance(x, np.ndarray):
        return np.where(x >= 0, 1, 0)
    else:
        return 1.0 if x >= 0 else 0.0 

x = np.linspace(-1, 1, 1000)

sign = np.sign(x)
step = heaviside_step(x)

plt.figure(figsize=(7, 4))
plt.subplot(121)
plt.plot(x, sign, "r-")
plt.title("부호 함수")
plt.ylim(-1.5, 1.5)
plt.subplot(122)
plt.plot(x, step, "k-")
plt.title("헤비사이드 스텝 함수")
plt.ylim(-1.5, 1.5)
plt.tight_layout()
plt.show()

연습 문제 4.2.3의 답¶

\[ \begin{align} f(x) = x^3 - 1 \tag{A.4.8} \end{align} \]
\[ \begin{align} f'(x) = 3x^2 \tag{A.4.9} \end{align} \]

(2)

\[ \begin{align} f(x) = \log (x^{2}-3k) \tag{A.4.10} \end{align} \]
\[ \begin{align} g(x)=x^2-3k \tag{A.4.11} \end{align} \]
\[ \begin{align} f(g)=\log g \tag{A.4.12} \end{align} \]

라고 하면

\[ \begin{align} f'(x) = \dfrac{df}{dg} \cdot \dfrac{dg}{dx} = \dfrac{1}{g} \cdot 2x = \dfrac{2x}{x^2-3k} \tag{A.4.13} \end{align} \]

(3)

\[ \begin{align} f(x) =\exp({ax^b}) \tag{A.4.14} \end{align} \]
\[ \begin{align} g(x)=ax^b \tag{A.4.15} \end{align} \]
\[ \begin{align} f(g)=\exp g \tag{A.4.16} \end{align} \]

라고 하면

\[ \begin{align} f'(x) = \dfrac{df}{dg} \cdot \dfrac{dg}{dx} = \exp{g} \cdot abx^{b-1} = abx^{b-1} \exp({ax^b}) \tag{A.4.17} \end{align} \]
